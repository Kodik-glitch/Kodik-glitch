01. Вывод данных

Используй print(), чтобы вывести что угодно в консоль справа.

print("Система активна")

02. Переменные

Это хранилища для данных. Числа пишутся просто так, текст — в кавычках.

power = 100
status = "Online"

03. Списки (Массивы)

Позволяют хранить набор элементов под одним именем.

tools = ["Браузер", "IDE", "Git"]
print(tools[0])

04. Условия

Если условие верно — выполняется один код, иначе — другой.

if power > 50:
  print("Работаем")
else:
  print("Зарядка!")

05. Циклы (Loops)

Повторяют код несколько раз. Цикл for идеален для перебора данных.

for x in range(5):
  print(f"Шаг №{x}")

06. Функции

Группируют код в блок, который можно вызвать по имени.

def start():
  return "Запуск..."

print(start())
